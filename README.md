# FilmKatalog


FilmKatalog je jednostavna web stranica koja omogućuje pregled popularnih filmova iz različitih žanrova.

## Stranice
- index.html - Naslovnica s izdvojenim filmovima i uvodom  
- about.html - Stranica s informacijama o projektu i njegovoj svrsi  
- katalog.html - Pregled svih filmova u katalogu  
- kontakt.html - Kontakt forma za korisnike  

## Kako pokrenuti
Otvori `index.html` u browseru ili koristi Live Server u Visual Studio Code za bolji pregled promjena u realnom vremenu.
